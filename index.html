Minesweeper is a classic game where the player aims to clear a board containing hidden "mines" without detonating any of them. The location of the mines can be determined through numbered clues displayed in adjacent tiles. If the player suspects that a tile contains a mine, the tile can be marked by placing a flag with the right mouse button. The game is won by clearing all tiles without hitting a mine.

In this Tiger King version, the same rules apply, however the mines are replaced by Joe Exotic and the flags are displayed as paw prints
